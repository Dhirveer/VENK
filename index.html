Course Information

This free online course will help you comprehend essential concepts and challenges associated with embedded systems.

Embedded systems design is a broad area that includes massive systems like MR scanners and small systems like chips. These systems have a lot of limitations that make it hard for those who are not familiar with the ideas to learn. The goal of this course is to assist you in fully comprehending all facets of the challenges and concepts linked to embedded systems. It starts with an explanation of what embedding and computing systems consist of.

The course will then introduce the main embedded systems challenges. Further, you will understand embedded systems implementation techniques, microcontroller processors, and CPUs. Instruction Set Architecture (ISA) and its main representatives, RISC and CISC, will next be discussed. Further, you will explore all the steps of the processor cycle in detail. The following example will make it even more transparent and easy to understand.

Next, you will comprehend the various types of memory used to store data. First, you will learn about volatile and non-volatile memory, NVRAM memory, and diverse Random Access Memory (RAM) and Read Only Memory (ROM) categories. The final section will discover different types of CPU architecture. You will understand the similarities and differences between Von Neumann and Harvard Architecture.
